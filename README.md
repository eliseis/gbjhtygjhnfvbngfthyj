# Internship Test Assignment

This repository contains my solution for the internship test assignment. The assignment was the same for all three positions I applied for:
- **Hallucination Detection**
- **Test Time Reinforcement Learning**
- **Learning to Reason with Small Models**

## Evaluation and Solution Files

There are two versions of the Jupyter Notebook with the solution in this repository. **I kindly ask you to evaluate the final solution — the `word2vec_sgns.ipynb` file.**

Here is why there are two files in the repository:
* **`word2vec_sgns.ipynb`** is the final version of my solution.
* **`word2vec_sgns_exp1.ipynb`** is an earlier version. The deadline for the "Hallucination Detection" position was earlier than the others, so I uploaded the intermediate version I had at that time. I deliberately left it in the repository just in case the specific deadline for each position is strictly enforced during the selection process.

Additionally, the model's artifacts are saved in the `artifacts/` folder:
* `word2vec_vocab.pkl` — vocabulary;
* `word2vec_weights.npz` — model weights.
